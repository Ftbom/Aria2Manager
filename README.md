# Aria2Manager

Aria2远程管理工具

## 功能

* 支持添加多个Aria2服务器
* 添加、暂停、继续、删除下载项
* 显示下载项详细信息，可选择要下载的文件
* 支持添加种子文件，支持HTTP连接基本设置
